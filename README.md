# 🔮 The Oracle - Fortune Teller

Uma aplicação web interativa e elegante que revela previsões aleatórias para o dia do usuário com o clique de um botão.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estrutura semântica e acessibilidade.
* **CSS3:** Variáveis nativas, Flexbox para responsividade e animações em `@keyframes`.
* **JavaScript:** Lógica de manipulação do DOM (Document Object Model) e algoritmos de geração de dados aleatórios.

## 🧠 Aprendizados
Este projeto foi desenvolvido com foco em:
1. **Refatoração de código:** Transição de estruturas condicionais `if/else` estáticas para estruturas dinâmicas utilizando Arrays de forma escalável.
2. **Manipulação do DOM:** Interatividade através de ouvintes de evento (`addEventListener`) controlando dinamicamente classes e conteúdos CSS.
